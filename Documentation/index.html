Introduction

Landsay is fully responsive minimal template specifically built for agencies, shop, store and personal website. It’s fully responsive and built with angular 17 and bootstrap v5.3.2. You will be easily able to customize it for your needs. It can be used to build a perfect website for any agency business, creative agencies, architecture, designer, photographers, personal etc. It’s minimal design with super simple code gives all flexibility to develop to build a great website.

Please feel free to get back to me in case if you are having any question or feedback.
